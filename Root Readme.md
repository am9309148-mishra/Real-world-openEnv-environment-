# 🚀 OpenEnv AI Delivery Simulator

An end-to-end reinforcement learning platform where an AI agent learns delivery optimization in real-time.

## 🔥 Features
- Custom RL Environment (Gym-like)
- DQN Training
- Live WebSocket Streaming
- Explainable AI
- Full-stack (Next.js + FastAPI)

## ▶️ Run

### Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm run dev
